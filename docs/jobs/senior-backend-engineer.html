Senior Backend Engineer

핀테크 스타트업 A • San Francisco, CA • Remote
알토스벤처스 투자 | Series B
상시채용

About the job

핀테크 스타트업 A는 혁신적인 금융 기술로 사용자들에게 더 나은 금융 경험을 제공하는 회사입니다. 우리는 Python/Django 기반의 글로벌 핀테크 플랫폼을 개발하며, 한국과 미국 시장을 동시에 서비스하고 있습니다.

이 포지션에서는 백엔드 시스템의 핵심 아키텍처를 설계하고 구현하게 됩니다. 대용량 트래픽을 처리할 수 있는 확장 가능한 시스템을 구축하고, 금융 규제 요구사항을 만족하는 보안 시스템을 개발하게 됩니다. 또한 한국과 미국의 개발팀과 긴밀히 협업하여 글로벌 서비스를 만들어 나가게 됩니다.

Minimum qualifications:

  • Python 개발 경험 5년 이상
  • Django 또는 Flask 프레임워크 숙련
  • AWS 클라우드 서비스 경험
  • 관계형 데이터베이스 설계 및 최적화 경험
  • RESTful API 설계 및 구현 경험
  • Git을 이용한 협업 경험

Preferred qualifications:

  • 핀테크 또는 금융 도메인 경험
  • Docker, Kubernetes 경험
  • 마이크로서비스 아키텍처 경험
  • 영어 업무 소통 가능
  • TDD 및 자동화 테스트 경험

Responsibilities

  • Python/Django 기반 백엔드 API 개발 및 유지보수
  • 글로벌 핀테크 서비스의 핵심 비즈니스 로직 구현
  • 대용량 트래픽 처리를 위한 시스템 아키텍처 설계
  • AWS 클라우드 인프라 구축 및 운영
  • 데이터베이스 최적화 및 성능 튜닝
  • 금융 규제 요구사항에 맞는 보안 시스템 구축
  • 한국-미국 팀과의 협업 및 코드 리뷰
