Frontend Engineer

이커머스 스타트업 B • Seattle, WA • Hybrid
스톤브릿지벤처스 투자 | Series A
지원 마감: 2024년 10월 15일

About the job

이커머스 스타트업 B는 글로벌 이커머스 혁신을 주도하는 빠르게 성장하는 스타트업입니다. 한국의 K-브랜드를 미국 시장에 소개하고, 미국의 우수한 제품을 한국에 도입하는 양방향 플랫폼을 구축하고 있습니다.

이 포지션에서는 React/Next.js를 활용해 글로벌 이커머스 플랫폼의 프론트엔드를 개발하게 됩니다. 사용자 경험을 최우선으로 하는 반응형 웹 애플리케이션을 구축하고, 결제 시스템과 장바구니 기능을 포함한 핵심 이커머스 기능들을 구현하게 됩니다.

Minimum qualifications:

  • React 개발 경험 3년 이상
  • TypeScript 숙련
  • 반응형 웹 개발 경험
  • CSS/SCSS 및 모던 CSS 프레임워크 경험
  • Git을 이용한 협업 경험
  • 웹 성능 최적화 경험

Preferred qualifications:

  • Next.js 또는 Nuxt.js 경험
  • 이커머스 플랫폼 개발 경험
  • GraphQL 또는 REST API 연동 경험
  • 영어 업무 소통 가능
  • 결제 시스템 연동 경험

Responsibilities

  • React/Next.js 기반 이커머스 웹 애플리케이션 개발
  • 반응형 웹 디자인 및 모바일 최적화
  • 사용자 경험(UX) 개선 및 성능 최적화
  • 결제 시스템 및 장바구니 기능 구현
  • 다국어 지원 및 국제화(i18n) 구현
  • A/B 테스트 및 데이터 기반 UI/UX 개선
  • 디자인 팀과 협업하여 디자인 시스템 구축
